# Pink Flower GitHub Pages Site

This repository contains a static web page (`index.html`) that displays a pink flower.

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, go to **Settings â†’ Pages**.
3. Under **Build and deployment**, select **Source: GitHub Actions**.
4. Ensure your default branch is `main` or `master` (the workflow listens to both).
5. Push changes and wait for the **Deploy static site to GitHub Pages** workflow to finish.

After deployment, your site will be available at:

`https://<your-username>.github.io/<your-repository>/`
